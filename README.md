C Programming Problems

This repository contains solutions to three basic C programming problems using conditional statements (if, else if, and else).

Problems Included
1. Odd or Even

Description:
Take an integer as input from the user and determine whether the number is odd or even.

Input:
An integer number.

Output:
"Even" if the number is divisible by 2, otherwise "Odd".

2. Pass or Fail

Description:
Take a student's marks as input and determine whether the student has passed or failed.

Input:
Marks obtained by the student (0–100).

Output:
"Pass" if the marks are 40 or above, otherwise "Fail".

3. Profit or Loss

Description:
Take the Cost Price (CP) and Selling Price (SP) as input and determine whether there is a profit, a loss, or no profit and no loss.

Input:

Cost Price (CP)
Selling Price (SP)

Output:

"Profit" if SP > CP
"Loss" if SP < CP
"No Profit No Loss" if SP = CP
